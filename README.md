# 📊 Employee Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing employee data to uncover insights related to workforce distribution, salaries, hiring trends, employee diversity, and attrition. The dashboard provides an interactive and visual way to explore HR data and support better business decisions.

## 🎯 Objectives

* Analyze employee distribution across departments and business units
* Understand salary structure and compensation trends
* Track hiring and employee exit patterns
* Identify workforce diversity insights
* Provide data-driven HR analytics for decision-making

## 📊 Dashboard Features

### 👨‍💼 Employee Workforce Analysis

* Total employee count
* Department-wise employee distribution
* Business unit performance analysis

### 💰 Salary & Compensation Analysis

* Annual salary comparison
* Bonus percentage insights
* Salary distribution trends

### 🌍 Diversity & Demographics

* Gender diversity analysis
* Ethnicity distribution
* Age group analysis

### 📅 Hiring & Attrition Trends

* Year-wise hiring trends
* Employee exit analysis
* Workforce growth tracking

### 📍 Location Analysis

* Country-wise employee distribution
* City-level workforce insights

### 🔍 Interactive Filters

* Department
* Gender
* Business Unit
* Country
* Year

## 🎨 Dashboard Design

* Clean and user-friendly layout

* Insight-driven visualizations
* Easy navigation and data exploration

## 💡 Use Cases

* Monitor workforce performance
